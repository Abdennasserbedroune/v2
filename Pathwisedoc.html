Project Overview

Pathwise is an AI-powered resume intelligence platform that transforms traditional CVs into actionable career insights. The platform serves two primary user groups:

  • Job Seekers: Receive detailed feedback on their resumes, including strengths, weaknesses, and improvement suggestions.
  • Recruiters: Compare multiple CVs efficiently and make data-driven hiring decisions.

Key Objectives

  • Automate the resume review process using AI/ML
  • Provide personalized, actionable feedback to job seekers
  • Streamline the recruitment workflow with intelligent comparison tools
  • Maintain a scalable and maintainable codebase

Frontend (Next.js 14)

The frontend is built with Next.js 14, leveraging the App Router for improved performance and developer experience. Key features include:

  • TypeScript for type safety
  • TailwindCSS for utility-first styling
  • React Query for data fetching and caching
  • Zod for runtime type validation

Database (PostgreSQL)

PostgreSQL serves as the primary database, storing:

  • User profiles and authentication data
  • Resume metadata and analysis results
  • User preferences and settings

Task Queue (Redis + Celery)

For handling long-running tasks like resume parsing and AI analysis:

  • Redis as the message broker
  • Celery for asynchronous task processing
  • Automatic retries and error handling

Core Features

1. Resume Upload & Parsing

Supports multiple file formats (PDF, DOCX) with intelligent parsing:

  • Extracts text, sections, and key information
  • Normalizes formatting and structure
  • Validates document content and completeness

2. AI-Powered Analysis

Advanced NLP and ML models provide comprehensive resume evaluation:

  • Skills extraction and categorization
  • Experience validation and ranking
  • ATS (Applicant Tracking System) compatibility scoring
  • Personalized improvement suggestions

Analysis Metrics

  • Content Quality: 95% accuracy in skills identification
  • Processing Time: < 10 seconds per resume
  • Format Support: PDF, DOCX, TXT

3. Resume Comparison

Recruiter-focused features for efficient candidate evaluation:

  • Side-by-side resume comparison
  • Skills gap analysis
  • Experience matching
  • Custom scoring criteria

Development Setup

Prerequisites

  • Node.js 18+ & npm 9+
  • Python 3.10+
  • Docker & Docker Compose
  • Git

Quick Start

# Clone the repository
git clone https://github.com/yourusername/pathwise.git
cd pathwise

# Start services using Docker Compose
docker-compose up -d

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
pip install -r requirements.txt

# Run database migrations
alembic upgrade head

# Start the development servers
# In separate terminals:
npm run dev  # Frontend
uvicorn app.main:app --reload  # Backend
celery -A app.worker worker --loglevel=info  # Celery worker

Challenges & Solutions

1. Resume Parsing Accuracy

Challenge: Extracting structured data from various resume formats with high accuracy.

Solution: Implemented a hybrid approach combining rule-based parsing with ML models, achieving 95% accuracy across common formats.

2. Performance Optimization

Challenge: Processing large PDFs and DOCX files efficiently.

Solution: Implemented streaming file processing and background task queues to handle large files without blocking the main application.

3. Cross-Platform Compatibility

Challenge: Ensuring consistent behavior across different operating systems and environments.

Solution: Containerized the entire application stack using Docker for consistent development and deployment environments.
